PIN block encoder and decoder

This web app calculates ISO 9564-1:2003 Format 0, 1, 2, 3 (ISO-0, ISO-1, ISO2, ISO3).

Fill the input field, then press the `Encode / Decode` button. For testing / verification purpose, you can use the `Prefill` buttons.
